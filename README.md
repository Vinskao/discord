### 使用說明

1. JDK version = 11;

2. 運行 ApiSpringBoot2Application.java 

      以啟動 Spring Boot

3. (optional)使用 postman 進行以下測試
   - 方法：GET method
   - 網址：http://localhost:8080/get-user-info/name-email/2
   - 備註：最後一個數字是想要查詢的人的 id

4. 打開http://localhost:8091/practice5/swagger-ui/index.html

      以查看api狀態
